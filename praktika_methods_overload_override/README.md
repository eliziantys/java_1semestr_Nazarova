# Тема: методы, перегрузка и переопределение

В этой папке:

- `Practice1.java` — пример перегрузки и переопределения методов

## Краткое объяснение

### Перегрузка методов

Перегрузка — это методы с одинаковым именем, но с разными параметрами.

В примере:

- `print(String text)`
- `print(int number)`

Это перегрузка, потому что имя метода одинаковое, а типы параметров разные.

### Переопределение методов

Переопределение — это изменение поведения метода суперкласса в подклассе.

В примере:

- в классе `Animal` есть метод `sound()`
- в классе `Dog` этот метод переопределен

Поэтому:

- `a.sound()` выводит звук животного;
- `d.sound()` выводит лай собаки.

# Тема: массивы

Примечание:

В задании указан класс `java.util.Array`, но в Java используется класс `java.util.Arrays`.

В этой папке:

- `Practice2.java` — пример методов `Arrays.toString(...)`, `Arrays.binarySearch(...)`, `Arrays.equals(...)`, `Arrays.compare(...)`, `Arrays.sort(...)`

## Практика 1

Дан код:

```java
int a1[] = {1,2,3,4,5};
int a2[] = {6,7,8,9,10};
a1 = a2;
```

Результат:

- массив `a1` перестанет ссылаться на свой старый массив `{1,2,3,4,5}`;
- после присваивания `a1` будет ссылаться на тот же массив, что и `a2`;
- теперь и `a1`, и `a2` указывают на массив `{6,7,8,9,10}`.

Это значит, что:

- `a1[0]` будет равно `6`;
- `a2[0]` тоже будет равно `6`;
- изменение элементов через `a1` будет видно и через `a2`, потому что это одна и та же область памяти.

Пример:

```java
a1[0] = 100;
System.out.println(a2[0]); // 100
```

## Практика 2

В файле `Practice2.java` показаны примеры следующих методов класса `Arrays`:

- `toString(...)` — превращает массив в удобную строку;
- `binarySearch(...)` — ищет элемент в отсортированном массиве;
- `equals(...)` — сравнивает массивы по содержимому;
- `compare(...)` — сравнивает массивы лексикографически;
- `sort(...)` — сортирует массив.

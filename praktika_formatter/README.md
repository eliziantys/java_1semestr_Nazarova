# Тема: java.util.Formatter

В этой папке:

- `Practice1.java` — пример использования пяти обычных спецификаторов форматирования
- `Practice3.java` — пример использования пяти спецификаторов для времени и даты

## Практика 1

В `Practice1.java` использованы пять спецификаторов:

- `%b`
- `%c`
- `%d`
- `%f`
- `%s`

## Практика 2

Метод `flush()` нужен для принудительной отправки накопленных данных из буфера в конечный поток вывода.

Это важно, когда:

- вывод буферизуется;
- нужно немедленно записать данные в файл, консоль или сетевой поток;
- программа еще не завершилась, но данные уже должны быть выведены.

Проще говоря, `flush()` заставляет систему "сбросить" буфер и реально записать все накопленное.

## Практика 3

В `Practice3.java` использованы пять спецификаторов времени и даты:

- `%tH`
- `%tM`
- `%tS`
- `%tY`
- `%tB`

## Краткий вывод

`Formatter` нужен для удобного форматированного вывода данных разных типов, включая числа, строки, логические значения, дату и время.

# Практика #1

Показана работа метода read() класса InputStream.

Использован ByteArrayInputStream.
Метод read() читает байты по одному.

# Практика #2

Показана работа метода write(int).

Использован ByteArrayOutputStream.
Метод write(int) записывает один байт.

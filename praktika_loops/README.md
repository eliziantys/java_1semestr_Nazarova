# Тема: циклы

В этой папке:

- `Practice1.java` — примеры бесконечных циклов `while` и `do-while`
- `Practice2.java` — пример использования `break` и `continue`

## Практика 1

Тривиальные примеры бесконечных циклов:

```java
while (true) {
    // бесконечный цикл
}
```

```java
do {
    // бесконечный цикл
} while (true);
```

В `Practice1.java` в оба примера добавлен `break`, чтобы программу можно было безопасно запускать.

## Практика 2

В `Practice2.java`:

- `continue` пропускает итерации с четными числами;
- `break` завершает цикл, когда `i == 7`.

То есть программа выводит только нечетные числа до момента остановки цикла.

Практика 3

Дан код:

```java
int a = 1;
{
    int b = 2;
}
int c = a + b;
```

Этот код не скомпилируется.

Переменная `b` объявлена внутри блока `{}`. Ее область видимости ограничена этим блоком, поэтому после выхода из блока переменная `b` больше не существует. Из-за этого выражение `a + b` вне блока использовать нельзя, и значение в переменной `c` не появится.

Практика 7

Дан код:

```java
int a = 120;
byte b = a + 10;
byte c = (byte)(a + 10);
byte d = a + 1;
```

Результат:

- `b` — ошибка компиляции
- `c` — `-126`
- `d` — ошибка компиляции

Объяснение:

- выражения `a + 10` и `a + 1` имеют тип `int`;
- переменной типа `byte` нельзя присвоить `int` без явного приведения;
- в строке с `c` используется приведение `(byte)`, поэтому код компилируется;
- число `130` не помещается в диапазон `byte` от `-128` до `127`, поэтому после приведения получается `-126`.

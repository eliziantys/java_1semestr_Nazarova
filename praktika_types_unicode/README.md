# Тема: базовые типы, Unicode, преобразование типов

В этой папке каждая практика вынесена в отдельный файл:

- `Practice1.java` — примеры всех базовых типов и нулевых значений
- `Practice2.java` — имя в формате Unicode
- `Practice3.java` — объяснение области видимости переменной `b`
- `Practice4.java` — перегруженный оператор `+` со `String`
- `Practice5.java` — арифметика с разными типами
- `Practice6.java` — сужающее преобразование типов
- `Practice7.java` — объяснение значений `b`, `c`, `d`
- `Practice8.java` — пример `interface` и `var`

Краткие пояснения:

- В практике 3 переменная `b` недоступна вне блока `{}`. Поэтому код не компилируется.
- В практике 7 переменные `b` и `d` нельзя инициализировать без явного приведения, потому что выражения имеют тип `int`.
- Значение `c` в практике 7 равно `-126`, потому что после приведения `130` к `byte` происходит переполнение диапазона.

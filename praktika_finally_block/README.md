# Практика #1

Конструкция try без catch и finally невозможна.
Такой код вызовет ошибку компиляции.

# Практика #2

Конструкция try + finally допустима.

Блок finally выполнится всегда,
даже если возникнет исключение.

# Практика #3

У одного блока try может быть только один finally.

Использование двух finally приводит
к ошибке компиляции.

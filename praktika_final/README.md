# Тема: final

В этой папке:

- `Practice1.java` — все основные варианты инициализации поля `final`

## Практика 1

Для поля

```java
public final int a;
```

возможны следующие способы инициализации:

### 1. Сразу при объявлении

```java
public final int a = 10;
```

### 2. В блоке инициализации объекта

```java
{
    a = 20;
}
```

### 3. В конструкторе

```java
public A(int value) {
    a = value;
}
```

## Важное замечание

Так как поле `a` является `final`, оно должно получить значение ровно один раз.

После инициализации изменить его уже нельзя.

Если поле `final` не инициализировать ни одним из этих способов, программа не скомпилируется.

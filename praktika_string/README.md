# Тема: класс String и StringJoiner

В этой папке:

- `Practice1.java` — демонстрация десяти методов класса `String`
- `Practice2.java` — примеры использования `StringJoiner`

## Практика 1

В `Practice1.java` показаны 10 методов класса `String`:

- `length()`
- `trim()`
- `toUpperCase()`
- `toLowerCase()`
- `charAt()`
- `substring()`
- `contains()`
- `equals()`
- `equalsIgnoreCase()`
- `replace()`

У каждого метода в коде есть короткий комментарий с пояснением.

## Практика 3

Три двойные кавычки нужны для **текстовых блоков** (`text blocks`).

Они позволяют удобно записывать многострочный текст без большого количества:

- символов `\n`;
- конкатенации строк через `+`;
- экранирования обычных кавычек.

Пример:

```java
String text = """
        Первая строка
        Вторая строка
        Третья строка
        """;
```

Это удобно, когда нужно хранить:

- большой текст;
- HTML;
- JSON;
- SQL-запросы.

# Тема: static

В этой папке:

- `Practice1.java` — пример вызова статического метода `printVars()`

## Практика 1

Для метода

```java
public static void printVars()
```

возможны следующие варианты вызова:

### 1. Через имя класса

```java
A.printVars();
```

Это правильный и рекомендуемый способ.

### 2. Через объект класса

```java
A obj = new A();
obj.printVars();
```

Такой вызов допустим, но использовать его не рекомендуется, потому что метод относится не к объекту, а к самому классу.

### 3. Через переменную-ссылку на объект

```java
A ref = new A();
ref.printVars();
```

Это тоже допустимо, но по той же причине считается нежелательным стилем.

## Вывод

Статический метод логически принадлежит классу, поэтому основной способ вызова — через имя класса:

```java
A.printVars();
```

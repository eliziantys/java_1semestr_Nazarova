# Тема: методы с переменным числом параметров

В этой папке:

- `Practice1.java` — пример класса, в котором перегружен метод с переменным числом параметров

## Практика 1

В примере создан класс `Printer`, где метод `print` перегружен в трех вариантах:

- `print(int... numbers)`
- `print(String... words)`
- `print(String title, int... numbers)`

Это перегрузка, потому что:

- у методов одинаковое имя `print`;
- но разные наборы параметров.

Такой пример показывает, что методы с переменным числом параметров действительно можно перегружать.

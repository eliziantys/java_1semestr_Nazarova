# Тема: перечисления enum

В этой папке:

- `Practice1.java` — пример использования инициализации элементов перечисления
- `Practice2.java` — пример собственного перечисления с дополнительным методом

## Практика 1

В `Practice1.java` создано перечисление `Season`.

Каждый элемент перечисления инициализируется через конструктор:

```java
WINTER("Зима", 3)
```

То есть каждому элементу передаются свои значения:

- название времени года;
- количество месяцев.

## Практика 2

В `Practice2.java` создано перечисление `TrafficLight` с дополнительным методом:

```java
public String action()
```

Этот метод возвращает действие для каждого сигнала светофора:

- `RED` -> `"Стой"`
- `YELLOW` -> `"Жди"`
- `GREEN` -> `"Иди"`

## Ограничения enum

Перечисления не могут:

- быть суперклассами;
- быть подклассами;
- быть параметризованными;
- создавать объекты через `new`.

Экземпляры перечисления создаются только внутри самого `enum` в виде фиксированного набора констант.

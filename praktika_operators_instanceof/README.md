# Тема: операторы и instanceof

В этой папке:

- `Practice1.java` — примеры операторов с базовыми типами и со `String`
- `Practice2.java` — примеры использования оператора `instanceof`

## Краткие пояснения

### Практика 1

В `Practice1.java` показаны:

- операторы присваивания `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- тернарный оператор `?:`
- логические операторы `||`, `&&`, `|`, `^`, `&`, `!`
- операторы сравнения `==`, `!=`, `>`, `>=`, `<`, `<=`
- сдвиги `>>`, `>>>`, `<<`
- арифметические операторы `+`, `-`, `*`, `/`, `%`
- префиксные и постфиксные формы `++` и `--`
- оператор `~`
- использование `( )` и `[ ]`
- конкатенация строк через `+` и `+=`

### Практика 2

Оператор `instanceof` возвращает `true`, если объект является экземпляром указанного класса, его подкласса или реализует нужный интерфейс.

Если переменная содержит `null`, то результат `instanceof` всегда будет `false`.

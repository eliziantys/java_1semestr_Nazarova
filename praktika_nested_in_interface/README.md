# Тема: вложенные классы в интерфейсе

В этой папке:

- `Practice1.java` — пример вложенного класса внутри интерфейса и способ вызова его метода

## Практика 1

В интерфейсе можно объявить вложенный класс.

Пример:

```java
interface Info {
    class Helper {
        public void show() {
            System.out.println("Метод вложенного класса в интерфейсе");
        }
    }
}
```

Такой класс вызывается через имя интерфейса:

```java
Info.Helper helper = new Info.Helper();
helper.show();
```

## Пояснение

Вложенный класс внутри интерфейса фактически является `public static`.

Поэтому:

- для его создания не нужен объект интерфейса;
- обращаться к нему нужно через имя интерфейса.

## Вывод

Если класс вложен в интерфейс, то его метод вызывается через объект, созданный так:

```java
InterfaceName.NestedClass obj = new InterfaceName.NestedClass();
```
